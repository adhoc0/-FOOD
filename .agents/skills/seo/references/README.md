# SEO References

## Purpose

This directory contains reference documentation for Search Engine Optimization (SEO) used in the FOOD project.

These documents provide implementation guidance, technical standards, optimization techniques, search engine recommendations, and best practices for developers and AI agents.

Project-specific SEO rules are defined in:

../SKILL.md

## File Naming

Use descriptive lowercase filenames.

Examples:

```text
technical-seo.md
keyword-research.md
structured-data.md
```

Avoid:

```text
notes.md
temp.md
misc.md
seo.md
new.md
```

---

## Guidelines

Reference documents should:

- Focus on one SEO topic.
- Be SEO-specific.
- Be concise and practical.
- Avoid duplicate information.
- Reference official Google Search Central and Schema.org documentation whenever appropriate.
- Be reviewed periodically.
