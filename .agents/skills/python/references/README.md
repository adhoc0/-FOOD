# Python References

## Purpose

This directory contains reference documentation for Python development used in the FOOD project.

These documents provide implementation guidance, language features, standard library usage, design patterns, coding conventions, performance optimization, and best practices for developers and AI agents.

Project-specific Python rules are defined in:

../SKILL.md

## File Naming

Use descriptive lowercase filenames.

Examples:

```text

```

Avoid:

```text
notes.md
temp.md
misc.md
new.md
python.md
```

## Guidelines

Reference documents should:

- Focus on a single topic.
- Be Python-specific.
- Be concise and practical.
- Avoid duplicate information.
- Reference official Python documentation (docs.python.org) whenever appropriate.
- Include code examples where useful.
- Be reviewed periodically.
