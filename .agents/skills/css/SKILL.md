# CSS

## Genel Kurallar

- Responsive tasarım zorunludur.
- Mobile First yaklaşımı kullanılmalıdır.
- CSS değişkenleri kullanılmalıdır.
- Magic Number kullanılmamalıdır.

---

## Tipografi

- Minimum font boyutu 16px
- Okunabilir satır yüksekliği kullanılmalıdır.
- Uzun satırlar sınırlandırılmalıdır.

---

## Focus

- Focus görünür olmalıdır.
- Hover ve Focus durumları birlikte tanımlanmalıdır.

---

## Performans

- Gereksiz selector kullanılmaz.
- Derin selector zincirleri kullanılmaz.
- Animasyonlar optimize edilmelidir.

---

## Yasaklar

- !important
- Inline Style
- Gereksiz selector
